# AidChain on Celo Network

This repository contains the codebase for a decentralized application (DApp) built on the Celo blockchain. The DApp aims to facilitate transparent and accountable charitable donations and aid distribution.

## Features

- User Registration and Onboarding
- Donations and Contribution
- Aid Distribution
- Real-Time Tracking and Reporting
- Decentralized Verification and Reputation
- Feedback and Testimonials
- NFT-based Fundraising and Memorabilia
- Community Governance
- Multi-Language Support
- Smart Contracts for Fundraising Events
- Privacy-Preserving Donation Options
- Smart Impact Metrics
- Inclusive Payment Options
- Disaster Relief and Emergency Funds

## Getting Started

### Prerequisites

- Node.js and npm
- Expo CLI
- Celo WalletConnect Provider
- Celo Wallet
- Solidity Smart Contracts

### Installation

1. Clone the repository: `git clone https://github.com/engrusmanbelloa/aidchain.git`
2. Navigate to the project directory: `aidchain`
3. Install dependencies: `yarn install`
4. Configure the WalletConnect provider and Celo wallet addresses in the code.

### Usage

1. Start the Expo development server: `yarn start`
2. Scan the QR code with your Expo Go app on your mobile device.
3. Follow the on-screen instructions to register, connect your wallet, and explore the features.
